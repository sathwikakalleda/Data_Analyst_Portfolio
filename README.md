# Sathwika Kalleda —Data Analyst Portfolio

A personal portfolio website showcasing my data analytics work, experience, skills, and certifications. Built with HTML, CSS, and vanilla JavaScript — no frameworks required.

## 🛠 Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure and content |
| CSS3 | Styling, layout, animations, responsive design |
| Vanilla JavaScript | Carousels, modals, navigation, scroll behaviour |
| Google Fonts | Plus Jakarta Sans (headings) + Inter (body) |
| Font Awesome 6 | Icons throughout |

No frameworks. No build tools. No dependencies to install.

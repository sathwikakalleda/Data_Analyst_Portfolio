### 🟡 Familiar with Git & Web Dev?
**Quick start:**

1. Add images to `assets/` folder
2. Edit `index.html` - update personal info
3. Test: Open `index.html` in browser
4. Push to Git:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin <your-repo-url>
   git push -u origin main
   ```
5. Deploy via GitHub Pages

## ❓ Common Questions

**Q: Do I need to know how to code?**  
A: No! Just follow the CHECKLIST.md guide. You'll mainly be updating text.

**Q: What if I get stuck?**  
A: Each guide has a "Troubleshooting" section. Most issues are covered.

**Q: Can I customize the colors/design?**  
A: Yes! Edit the CSS variables in `css/style.css` (instructions in SETUP_GUIDE.md)

**Q: Is this free to use?**  
A: Yes! MIT License. Use it however you want. Deploy for free on GitHub Pages.

**Q: What if I don't have Git?**  
A: Download from [git-scm.com](https://git-scm.com) or use [GitHub Desktop](https://desktop.github.com)

**Q: Will this work on my domain?**  
A: Yes! Works with any hosting. Instructions in DEPLOYMENT.md
